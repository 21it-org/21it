    "Hello World"

    はじめに

    "Hello world"はプログラムを学ぶ際に出会うサンプルプログラムで表示される代表的な文言ですが、ごく単純な表示プログラムが動いたということがそれが表示されることで確認できます。一方で、それはあたかもコンピューターが語りかけてくるようなどこか不思議な響きでもあります。ITテクノロジーを学んでいく中で、こうした機知にとんだ表現や概念に出会うことが多くあります。本サイトは21世紀IT塾のオンラインドキュメントとして、ITテクノロジーの教育コンテンツを提供しています。先人たちが作り上げた機械の世界の深淵を垣間見、学んでいいただければ幸いです。21世紀IT塾の理念と目的についてはホームページを参照してください

    8つの学習コンテンツ

    本サイトでは8つの学習コンテンツを提供しています。それぞれ自分のペースで無理なく学んでいただければと思います。

    Course Description
    UNIX基礎#1 学習環境であるUNIX/Linuxの概要と基本的なコマンドを紹介します
    UNIX基礎#2 Unixが作り上げた高度なコマンドとそれをパイプなどでつなげていく技法を紹介します
    shellプログラミング 基本的なコマンドをならべるだけで即席のプログラミングをくみ上げます
    インターネット基礎 インターネットの原理的な仕組みについて紹介します
    small-shell基礎#1 small-shellをもちいたアプリケーション生成とデータ管理の基礎を説明します
    small-shell基礎#2 small-shellを用いて様々なタイプのWebアプリを生成します
    small-shell応用 small-shellにコードを追加し独自のアプリを作成する方法を紹介します
    デジタル耐性 21世紀のあるべきITリテラシー(Digital耐性と呼称)について考えます
